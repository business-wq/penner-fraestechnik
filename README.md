# Marklewitz Solar

Website für Marklewitz Solar – Photovoltaik & Solaranlagen in der Region Kirchheim unter Teck, Owen, Nürtingen und Esslingen.

## Tech Stack

- **Vite** + **React** + **TypeScript**
- **React Router v6**
- **Tailwind CSS** + **shadcn/ui**
- **Framer Motion**

## Entwicklung

```bash
npm install
npm run dev
```

Dev-Server: http://localhost:5173

## Build & Deploy

```bash
npm run build
```

Output in `dist/`. Die Website wird bei Hostinger gehostet.

## Befehle

| Befehl | Beschreibung |
|--------|--------------|
| `npm run dev` | Dev-Server starten |
| `npm run build` | Production-Build |
| `npm run preview` | Build lokal testen |
| `npm run lint` | ESLint ausführen |
| `npm run test` | Vitest (Tests) |
